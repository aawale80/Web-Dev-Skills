WEB DEVELOMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Language. HTML is the content of a web page. Examples of HTML include headings, paragraphs, images, links, lists, etc

Check out the HTML Tutorial on w3schools.com

Learn CSS

CSS stands forCascading Style Sheets. CSS is thestyle of web page. Examples of CSS includes font styles and color, background color, borders, dimensions, layout, etc.

Check out the CSS Tutorial on w3schools.com

Learn Javascript

Javascript is the programming language of the web. Javascript helps to make web pages interactive and dymatic. Some common Javascript tasks are:

  • add new HTML to the page,
  • change the existing content, / style,
  • react to user events (mouse click, key presses, etc.)

Check out the Javascript Tutorial on javascript.info
